Project 1 Title

Repository: github.com/tusssars11/project1

Description:
This project is a D3.js data visualization dashboard that displays real-time metrics for a simulated sensor network. It features interactive filtering, tooltips, and a responsive layout that adapts to mobile and desktop. By leveraging WebSockets, the dashboard updates live without requiring a full page refresh.

Tech Stack:
D3.js, JavaScript (ES6+), HTML5, CSS3, Node.js, Express.js, MongoDB, Socket.io.

Key Features:

  • Real-time charts: Built using D3.js with smooth transitions on incoming data.
  • WebSockets integration: Used Socket.io to push sensor updates from Node.js server to front-end in real-time.
  • Responsive design: Automatically adjusts to various screen sizes using CSS Grid and Flexbox.

Timeline: January 2023 – March 2023
Role: Full-Stack Developer (designed front-end D3 components & implemented back-end API).
