Portfolio Website Launch

After weeks of planning and design, I built my new portfolio site completely from scratch using plain HTML, CSS, and vanilla JavaScript—no frameworks. The goal was to keep it lightweight, fully responsive, and showcase both my code samples and my interactive projects (like the D3 dashboard).

Key Highlights:

  • Responsive Design: Adaptable layout that works on mobile, tablet, and desktop using CSS Flexbox and media queries.
  • Animated Hero Section: Simple CSS‐only animations on page load to draw attention to my headshot and tagline.
  • Project Showcase: A grid section displaying recent projects with hover‐zoom effects (using pure CSS transforms).
  • Dark/Light Mode Toggle: Built a JavaScript toggle that remembers your preference (using localStorage).

Tech Stack & Deployment

• HTML5, CSS3, JavaScript (ES6)
• No build step—everything is static HTML/CSS/JS
• Hosted on GitHub Pages at tusssars11.github.io
